創新環保綠色搬家-伸縮防護套

與物品打包同樣是搬家重點的，則是傢俱與家電的搬運與防護。按照傳統的搬家習慣，經常對傢俱家電幾乎不做防護就直接搬上車，稍講究一些的，才會利用大量的氣泡紙、膠膜、瓦楞紙、膠帶等耗材包裝，才能達到保護的效果，而且搬遷完畢包材得拆掉丟棄，製造可觀的垃圾。而現在環保搬家推廣的伸縮防護套、軍毯與扣帶，不但可以應用在任何尺寸的傢俱家電上，並且由於容易包裝，防護功能又優於傳統耗材，有效預防傢俱損傷，也減輕環境負擔。 伸縮防護套材質具有伸縮性，也有一定厚度，幾乎套得下任何尺寸的傢俱，也使傢俱得到良好的防護與緩衝的效果，同時也讓搬家師傅的包裝作業便利順暢。而取得容易的軍毯，與防護套同樣可多次重覆使用，防護功用也不遜於氣泡紙或膠膜等耗材，是伸縮防護套普及前很合適的包材。與上述兩種包材搭配的扣帶，則可取代以前黏綁用的

搬家知識
Q
如何預約呢?
一般遊客無須預約，直接購票入場即可。
30人以上(含)可預約，請電洽03-9907101 或至聯絡我們填寫表單
Q
一般遊客與團體行程有何差別?
行程完全相同。
Q
可以僅陪同不參加DIY嗎?
無論大人或孩童，進館皆須收取門票費用，門票即包含DIY體驗，不可單獨計價。
為確保館內兒童安全，入館時，兒童皆須有大人陪同，且兩者亦無法單獨購票入場。
Q
可以免費參觀嗎?
本館大廳設有常態產覽，並有商品販售區，歡迎免費參觀；
DIY區、專案體驗與彩虹溜滑梯則需購票入場。
Q
幾歲適合入館呢?
蜡藝蜡筆城堡歡迎所有年齡層的大朋友小朋友，3歲以上孩童會比較了解DIY內容。
(3歲以下與65歲以上長者可免購票入館，無DIY體驗)

常見問題
Q
可以攜帶食物、飲料入內嗎?
可攜帶飲食入DIY區，勿佔用DIY桌椅，飲食以不影響其他遊客為原則。
(禁止攜帶榴槤或臭豆腐等味道重的食物；禁止攜帶酒精類飲料進入。)
餐飲區全區禁帶外食。
Q
寵物可以入館嗎?
館內實施寵物不落地，中小型寵需置於寵物袋或推車內即可。(導盲犬除外)
Q
可以中途離開再折返嗎?
可以，憑當天票券皆可重複入場，但每項DIY僅可參加一次。
出館之後須再次入場請與櫃檯人員報到。
Q
停車場需要付費嗎?可否預留車位?
本館設有120個免費停車位。(停車場僅供本館遊客使用，非本館遊客不可停放)
遊覽車若需預留車位請電洽03-9907101 。
Q
館內是否提供儲物空間?
本館提供免費置物櫃與推車停放區，使用請洽櫃台人員。
個人財務請自行善盡保管之責，若有遺失本館恕不負保管責任。
